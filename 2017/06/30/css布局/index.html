float属性以及三列布局

发表于 2017-06-30 | 分类于 Css | | 本文总阅读量 次

##设置float属性，父元素高度会变成0解决方案
（float会使元素脱离文档流。）

1、在浮动元素下面加一个清除浮动的元素:div{clear:both}

html

<div class="clearfix">
    <div style="float: left;">Div 1</div>
   <div style="float: left;">Div 2</div>
</div

css

.clearfix:after { 
   content: " ";
   display: block; 
   height: 0; 
   clear: both;
}

2、将父元素也设置浮动

html

<div style="float: left;">
    <div style="float: left;">Div 1</div>
   <div style="float: left;">Div 2</div>
</div

3、为父元素加入overflow:hidded属性或者overflow:auto属性

html

<div style="overflow: hidden;">
    <div style="float: left;">Div 1</div>
   <div style="float: left;">Div 2</div>
</div

float在什么情况下失效

1、两个同级div，第一个设置float:left,第二个未设置，则会第一个脱离了文档流，也就是说第二个div无视第一个的存在，将与之发生重叠

2、absolute存在的时候float会失效

##三列布局

1、双飞翼布局

html

<div class="mydiv">
    <div id="middle">
        <div id="middle-wrap"><span>div-middle</span></div>
    </div>
    <div id="left"><span>left</span></div>
    <div id="right"><span>right</span></div>
</div>

style

#middle {
    float: left;
    background-color: #fff9ca;
    width: 100%;
    height: 50px;
}
#left {
    float: left;
    background-color: red;
    width: 150px;
    margin-left: -100%;
    height: 50px;
}
#right {
    float: left;
    background-color: yellow;
    width: 200px;
    margin-left: -200px;
    height: 50px;
}
#middle-wrap {
    margin: 0 200px 0 150px;
}

效果预览

2、绝对定位布局

html

<div class="mydiv">
    <div class="left"></div>
    <div class="right"></div>
    <div class="middle"></div>
</div>

style

.mydiv{
    background-color: #eee;
    margin:20px;padding: 20px;
    border: solid 1px #999;
    position: relative;
}
.left {
    width: 160px;height: 100px;padding: 20px;
    background: blue;
    position: absolute;left: 20px;
}
.right{
    width: 80px;height: 260px;padding: 20px;
    background: blue;
    position: absolute;right: 20px;
}
.middle{
    background: red;
    height: 200px;padding: 20px;
    margin-left: 220px;margin-right: 140px;
}
[效果预览][2]

2、浮动布局

html

<div class="mydiv">
    <div class="left"></div>
    <div class="right"></div>
    <div class="middle"></div>
</div>

style

.mydiv{
    background-color: #eee;
    margin:20px;padding: 20px;
    border: solid 1px #999;
    overflow: auto;
}
.left {
    float: left;
    width: 160px;height: 100px;padding: 20px;
    background: blue;
}
.right{
    float: right;
    width: 80px;height: 300px;padding: 20px;
    background: blue;
}    
.middle{
    margin-left: 220px;margin-right: 150px;
    height: 200px;padding: 20px;
    background: red;
}
[效果预览][3]

4、 flex布局

html

<div class="flex">
    <div class="left"></div>
    <div class="middle"></div>
    <div class="right"></div>
</div>

style

.flex {
    display: flex;
    flex-flow: row;
}
.left {
    background-color: red;
    width: 150px;
    height: 50px;
}
.middle {
    background-color: #fff9ca;
    flex:1;
    height: 50px;
}
.right {
    background-color: yellow;
    width: 200px;
    height: 50px;
}
（在 flex 布局中不能将被定位的元素宽度或高度设定为 100%，这样会影响其他定值大小的元素。可设置为flex: 1;）
